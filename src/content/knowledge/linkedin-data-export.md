# LinkedIn人才搜索与管理操作指南

## 项目背景与目标

### 核心目标
- **主要任务**：通过LinkedIn收集目标人才信息
- **重点方向**：专注于海外华人（Chinese）人才
- **预期比例**：约70-80%的人可以找到并查看相关信息
- **关键词示例**：LLM等相关技术领域

## 对LinkedIn搜索操作流程

### 账号绑定准备
1. 打开浏览器开发者工具（F12）
2. 找到XHR（XMLHttpRequest）选项
3. 复制相关权限代码
4. 授权给操作人员进行搜索

### 搜索策略

#### 关键词组合方法
- **基础关键词**：LLM
- **职位关键词**：Engineer、Manager、Researcher等
- **教育背景**：Master,PHD等
- **变体组合**：尝试不同拼写和缩写方式

#### 搜索条件设置
```
- 产品/技术：LLM
- 地点：美国
- 职位类型：相关技术岗位
- 学历背景：硕士/博士优先
```

### 结果优化技巧

**重要限制**：
- 运营账号搜索限制：只能查看前1000个结果
- **解决方案**：必须拆分搜索条件

**拆分示例**：
- 添加具体职位名称（Engineer、Manager等）
- 限定教育背景（Computer Science等）
- 通过关键词组合缩小到200-300人的范围

## 项目管理与数据保存

### 创建项目
1. 新建项目
2. 输入搜索条件
3. 选择地点（美国）
4. 保存搜索结果到项目中

### 数据获取注意事项

**关键原则**：
- ⚠️ **一个一个执行任务**，不要同时启动多个
- 模拟真实人工操作，避免被识别为机器
- 如遇意外终止，可在底部继续执行
- 等待一个任务100%完成后再进行下一个

**获取进度**：
- 系统会显示下载进度（如40%、100%）
- 可实时查看已获取的人员数量
- 完成后可导出Excel表格

### 华人识别功能
1. 使用系统自动识别功能
2. 准确率约70-80%
3. 可手动标注遗漏的华人
4. 创建"华人"专项分类

## 人才评估与筛选

### 查看候选人信息
**可获取内容**：
- 基本信息（姓名、头像）
- 教育背景（学校、专业、学位、年份）
- 工作经历（公司、职位、项目）
- 技能关键词
- GitHub等其他平台链接


### 联系方式获取

**多渠道获取**：
1. **LinkedIn直接获取**：部分公开邮箱
2. **GitHub查找**：
   - 查看贡献记录
   - 在代码提交中找公司邮箱
   - 注意：需公开信息才能获取
3. **个人主页**：查看是否有联系方式
4. **简历链接**：部分候选人会附简历

**注意事项**：
- 有些人会隐藏邮箱
- 优质人选值得花时间查找
- 非优质人选可跳过

## 数据管理与协作

### Excel表格管理

**统一模板字段**：
- 基本信息
- 职位
- 地点
- 公司
- 部门
- 教育背景
- 项目介绍
- 搜索关键词

### 团队协作规范

**工作流程**：
1. 每人在群空间建立自己的Excel
2. 记录所有Chinese候选人
3. 标注是否获取邮箱
4. 记录回复情况（动态更新）

**资产积累**：
- 过程中的所有Chinese名单都要保留
- 作为团队资产存档
- 即使项目变更，数据仍可复用

### 去重与整理
- 系统会自动跨查询去重
- 手动标注重复出现的优质候选人
- 持续优化搜索关键词

## 最佳实践与技巧

### 关键词迭代
- 从候选人简历中提取新关键词
- 测试不同关键词组合
- 记录高质量关键词到表格
- 用于后续反向搜索

### 效率优化
- 先浏览结果判断质量
- 优质搜索才创建项目
- 边浏览边加入项目
- 利用华人识别功能快速筛选

### 持续迭代
```
初始搜索 → 发现优质人才 → 
提取关键词 → 新一轮搜索 → 
发现更多人才 → 持续循环
```

## 注意事项总结

### ⚠️ 关键提醒
1. **不要批量操作**：一次一个任务，模拟真人行为
2. **控制搜索规模**：保持在1000人以内
3. **精准搜索**：通过关键词组合缩小范围到200-300人
4. **完整记录**：所有Chinese候选人都要记录在Excel
5. **资产意识**：数据是团队长期资产
