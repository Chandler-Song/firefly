# Duolaimeng平台使用指南

## 网址

网站：www.duolaimeng.online （请登录该网站进行注册&登录）

---

## 工作原理

登录LingYing账号，利用账号的Cookie来代理访问（关于什么是cookies可以自行百度），本质上跟使用LingYing账号没什么区别。但相比其他一些需要输入账号和密码的第三方工具，用Cookie相对安全，保证账号密码不会被泄露，因为LingYing网站会不定时更换Cookie，如果失效了，再重新登录账号获取Cookie即可。
